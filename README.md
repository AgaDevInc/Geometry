# @agadev/geometry
## A simple package to add colors to your terminal

## Imports
### Deno
```js
import * as geometry from 'https://agacdn.onrender.com/agadev:geometry';
```
### CommonJS
```js
const colors = require('@agadev/geometry');
```
### ECMAScript
```js
import * as geometry from '@agadev/geometry';
```
